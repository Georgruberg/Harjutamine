Githubi kasutatakse versioonihalduseks ettevõtte või projekti siseselt, et hõlbustada mitme inimese koostööd sama projektiga.
Selles saab liikuda muudatuste ajaloos tagasi ning uuendused panna üle vaatamiseks valmis enne kui nad reaalselt töösse pannakse.

Kuna osalen Java programmeerimise kursusel nädalavahetustel, siis hakkan seda kasutama grupitöödes ning kui saan tööd tarkvara arenduses siis ka tööjuures.

GitHub on Ameerika firma, mis pakub veebimajutusteenust tarkvaraarenduseks, kasutades Giti tarkvara. GitHub on Microsofti tütarfirma alates aastast 2018, mil Microsoft ostis GitHubi 7,5 miljardi dollari eest. GitHub põhineb Giti hajutatud versioonihaldusel ja lähtekoodihaldusel, kuid lisab neile veel võimalusi, näiteks võimalust kontrollida, kes pääseb koodile juurde, koodi veakontrolli, tarkvara funktsioonide taotlust ning igale projektile oma vikit. GitHub on kirjutatud Rubys ning baseerub Ruby on Railsi raamistikul.
